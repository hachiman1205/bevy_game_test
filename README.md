# My Bevy Game

A simple, lightweight 2D/3D game built in Rust using the **Bevy Engine**. 

---

## 🚀 Features

* **Fast & Modern:** Built on top of the data-driven Bevy ECS (Entity Component System).
* **Cross-Platform:** Works on Windows, macOS, and Linux.
* **Hot Reloading:** Asset hot-reloading enabled for fast development iterations.

---

## 🛠️ Prerequisites

Before running the game, make sure you have the following installed:

1. **Rust Toolchain:** Install via [rustup](https://rustup.rs/).
2. **System Dependencies:** Bevy requires certain graphics libraries depending on your OS.
   * **Ubuntu/Debian:** `sudo apt install pkg-config libx11-dev libasound2-dev libudev-dev libwayland-dev libxkbcommon-dev`
   * **macOS/Windows:** No extra system dependencies typically required.

---
